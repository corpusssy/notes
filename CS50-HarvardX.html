Theory

  • Computers only speak Binary - A combination of 0s and 1s
    • By using different patterns of 0s and 1s, these binary digits (bits) can be used to represent different numbers and letters
    • Computers only speak Binary because it's easier to store

    • Binary digits lengths:
      • Bit - A binary number (Either 0 or 1)
      • Byte - A block of 8 bits
      • Gigabyte -
      • Terabyte -
      • etc.

    • How does a computer map which bits represent letters, numbers, characters, etc.?
      • By having different file formats, containers, etc.
      • By leaving hints to the computer that tells it to interpret the following bits as follows

  • ASCII - American Standard Code for Information Interchange

  • Unicode - A mapping (A collection of values used as input) of many more numbers to many more numbers, letters and characters
    • Allows you to represent more than 256 characters (one byte per character), such as letters with accents and emojis
    • Unicode can use 16 bits, 32 bits, etc. which allows us to represent even more numbers, letters and characters

  • Computers represent:
    • Colors by using RGB values
    • Videos through Binary code (Since they're just a grid of pixels or dots that each are represented through Binary code and when brought together, form together an image and when put together in a sequence, creates a video
    • Music by repsenting each note as a sequence of numbers
    • etc.

  • Input => Algorithm => Output
    • Algorithm - Step-by-step instructions for solving problems

    • Arguments => Functions => Output

  • Pseudocode - A way of expressing yourself towards an end of getting your idea for your program accross
  • Events - Anytime you, a human, perform an action that the computer or program can listen for and repsond to

  • Basic universal syntax convetions:
    • Correctness - Making sure your code is correct
    • Design - Making your code nice, clean and efficient (Doesn't use too much memory and isn't redundant)
    • Style - Making your code look pretty to look at

  • How do I write code?
    • By using an IDE (Integrated Development Environment) to write your source code adn then use a compiler (a program who's purpose is to convert one language to another) to translate youe source code into machine language

  • What's a terminal?
    • It's a CLI (Command Line Interface) on top of an operating system that allows you to interact with a computer and run commands on a server that you have access to
    • It's where you'll be able to type in commands

  • Variables and functions

  • Loops
    • For loops
      • i is declared inside the for loop
    • While loops
      • i is declared outside the while loop
  • Integer Overflow
    • Integers can overflow if you're trying to use more bits than you have available (Each computer have limited bits)
    • Bits are small units of machine (Binary) code in a computer that holds data

  • Truncation
    • When you use integers together in any mathimatical equation, you'll get an integer as an answer but if one of the numbers involved is an float, the program will automatically throw away all the numebrs after the decimal point (Present in C, etc.)
    • To solve prevent this, you do a type conversion (int to float)

  • Syntax
    • You have to tell the computer what the data type of each variable in your program is, as well as what data type your function will return
    • Once you initialize a variable, you don't need to keep specifying it's data type

  • Operators
    • = - Assignment operator
    • % - Format string
    • +
    • -
    • /
    • *

  • Data types:
    • char - Single characters
    • double - A more precise number containing more numbers after the decimal point
    • float - A number with a decimal point in it
    • int - A whole number
    • long - A bigger whole number
    • void
      • - Not a data type but rather just a type
    • Additional data types (from the CS50 library)
      • bool - boolean (true or false)
      • string - An array of characters

  • Format strings
    • %c - char
    • %f - float or double (You can add the number of decimal numbers that you specifically want to have by adding a period with the number of decimals you want between the "%" and "f" - ex: %.2f)
    • %i - int
    • %li - long
    • %s - string
    • etc.
